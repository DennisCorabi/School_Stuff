#Creautenti.py:
Questo script crea una serie di utenti in automatico prelevati da un file di testo, che può essere modificato, purchè seguendo una serie di regole (LEGGI SOTTO).
Per quanto riguarda la struttura del programma, nel sorgente sono presenti, in inglese, dei commenti che spiegano il funzionamento delle varie parti del programma.

Il programma è semplice: dato un nome e password prelevati da un file di testo, crea in automatico un utente con le credeziali prelevate.

#utenti.txt
questo il file di testo da cui CREAUTENTI.PY preleva gli username degli utenti che si vuole creare e le rispettive password.

questo file di testo è liberamente modificabile (ovvero possono essere aggiunti altri utenti da creare),  purchè si rispetti un detemrminato formato:

FORMATO DEL TESTO DA INSERIRE: <Nome> <Password> 
Dopo il nome e dopo la password DEVE essere inserito uno spazio.
